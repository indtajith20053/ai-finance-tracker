# AI Finance Tracker

This is a simple AI-powered personal finance tracker built using Python, Gradio, and Ollama.

## Features
- Track income
- Log expenses
- Chat-based interface

## Tech Stack
- Python
- Gradio
- Ollama (LLaMA 3)

## How to Run
1. Install dependencies
2. Run ollama serve
3. Run notebook
